## 二、月省金额计算公式

```
月省金额 = Σ(各岗位人数 × 该岗位月薪 × 该岗位降本比例)
```

### 示例：10人团队（设计2人 + 运营3人 + 客服3人 + 开发2人）

| 岗位 | 人数 | 月薪 | 降本比例 | 月省金额 |
|------|------|------|---------|---------|
| 设计 | 2 | ¥10,000 | 80% | ¥16,000 |
| 运营 | 3 | ¥8,000 | 70% | ¥16,800 |
| 客服 | 3 | ¥6,000 | 80% | ¥14,400 |
| 开发 | 2 | ¥20,000 | 75% | ¥30,000 |

## 三、年度陪跑回本计算

### 回本公式

```
回本周期（月）= 98,000 ÷ 月省金额
```

## 六、成本计算器使用说明

前端成本计算器位于 JellyAssistant 组件的「🧮 成本计算器」Tab。

### 输入项
- 设计岗人数
- 运营岗人数
- 客服岗人数
- 开发岗人数

### 输出项
- 月省金额
- 年省金额
- 年度陪跑回本周期

### 计算逻辑
1. 月省 = 设计人数×10000×0.8 + 运营人数×8000×0.7 + 客服人数×6000×0.8 + 开发人数×20000×0.75
2. 年省 = 月省 × 12
3. 回本 = 98000 ÷ 月省（保留1位小数）

## 七、话术中的成本计算引用

当用户问"贵不贵"时，用以下话术：

```
算一笔账：你团队[X]人，AI替代后月省[Y]万，年省[Z]万。
98,000元的陪跑费，[N]个月就回本。
温州案例：50人→30人，年省120万+。
```

不承诺具体收益，只提供基于降本比例的估算参考。
